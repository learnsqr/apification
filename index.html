Seminario de Desarrollo de API's para Aplicaciones en PHP.

Se presentan de manera amplia los conceptos que ha de tener una API para trabajar con las diferentes partes que la consumen hoy por hoy. Hablaremos de los diferentes esquemas de seguridad que intervienen entre dichas partes y como se implementan usando las tecnologías del mercado PHP y Zend Framework para desarrollo de APIs.
